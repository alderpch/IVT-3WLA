# Examples

This folder includes example data sets and worked examples of IVT-3WLA analysis.

## Purpose

- To show how ACF, FC, and CVF behave with real 3WLA historical records
- To provide sample calculations
- To test the sensitivity of the IVT-3WLA under different conditions

## How to contribute

Users may contribute their own 3WLA history files through Pull Requests.
Please place your file inside this folder and follow the contribution guide in the main README.
