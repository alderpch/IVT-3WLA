# User Contributions

This folder is dedicated to user-submitted 3WLA historical datasets and analyses.

## How to contribute

1. Fork this repository  
2. Add your file (Excel, CSV, or ZIP) inside the `/users` folder  
3. Include a brief description in your Pull Request  
4. Submit the Pull Request  
5. Your contribution will be reviewed and, if approved, added to the project

## Guidelines

- Do not include confidential or proprietary information  
- Remove sensitive project identifiers  
- Use clear file names such as:  
  `3WLA_ProjectX_Sample01.xlsx`  
  `UserDataset_YourName.xlsx`

Thank you for helping expand the IVT-3WLA dataset.
